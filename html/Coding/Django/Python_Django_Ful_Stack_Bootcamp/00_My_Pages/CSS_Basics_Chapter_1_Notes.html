CSS Basics Chapter 1 Notes:

1.1: Basic CSS and Font Colors

We're going to make an HTML list below, and style it in different ways with CSS.

We do this by using a link with rel=stylesheet and href="" indicating the .css file to point towards.

By typing an HTML element such as "h1" or "li" followed by a "{" we can then insert certain properties for that element.

A list of example properties that can be changed with CSS:

  • "color:" is an example property that can change the color of text.
  • "color:" can also be set using a rgba value to set a color with Red, Green, and Red values from 0 to 255, and a number between 0 and 1 for alpha.
  • "font-size:" is another property that can be modified with values of small, medium, large, x-large, and xx-large
  • "font-family:" can indicate many properties aobut the font, weight, spacing, etc.

1.3: Selectors, Classes, IDs, and Elements

Selectors in general are used to target HTML elements that we want to style, similar to what we've done so far. This takes all li elements, h1 elements, etc. and selects them to be styled in a specific way.

We can IDs to target single elements and use classes to target groups of elements.

You can use *{} as a selector in CSS to override quite a few things, but this is fairly uncommon. The CSS for this page has done this to change font-weight to 750/1000. Tags with IDs will not be impacted.

You can also use a "+" to impact multiple selectors at a time. For example, we've now modified all ul+h3 items (unordered lists near/under h3 header) to have a 4px purple dotted line, as you now see in Section 1.1.

Similarly, we've now made all "li a" tags (anchor tags in list items) become the color red, as you see above with Facebook and Google.

However, we can partially override that by defining that all "li a[href="www.google.com]" tags have a black color, as we've done with the Google link above.
