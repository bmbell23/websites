HTML Basics Chapter 1:

Summary:

The Front End of websites are mostly comprised of 3 parts:

  1. HTML: Hyper Text Markup Language is a scripting language, and the main structure and contents of the page. This defines what text goes where, hyperlinks, images, tables, etc.
  2. CSS: Cascading Style Sheets defines the formatting and style of the HTML contents. This is what will define fonts, colors, etc.
  3. JavaScript: JavaScript allows for interactive elements of the website with the Back End. This is what allows for uploads, downloads, searches, etc.

The Back End of websites are also mostly comprised of 3 parts:

  1. Programming Language: Coding languages such as Python or Rust.
  2. Frameworks: Connect the Development language to the front end. Django is what we'll use.
  3. Databases: Store all the data in the background, and are accessed from the coding language.

1.1: Headings, Ordered Lists, Unordered Lists, Bold, and Italics

  1. Headings are added by using <h1> </h1> for h values from 1 to 6.
  2. Ordered lists can be made by using <ol> </ol>.
  3. Unordered lists can be made by using <ul> </ul>.
  4. List items on an ordered or unordered list are added with <li> </li>.
    • Lists may also be nested by simply nesting an ordered or unordered list within a list.
  5. Font is made Bold with <b> </b> or made Italic with <i> </i>
  6. Lorem Ipsum text can be inserted with simply typing "lorem"

Lastly, simple paragraph text can be added with <p> </p>.

1.2: Divs and Spans

Div tags can be added with <div> </div> and would have a specific class defined in them as seen below. These div tags can specify certain attributes to the class indicated to later define things like font and color.

For example:

<div class="groupone1">

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Esse quisquam hic illum unde enim id repellendus, voluptate doloribus adipisci ea dolore consequatur, consectetur neque nisi vel non sequi tenetur? Magni!

</div>

Span tags however, can be added to specific text in the middle of a paragraph using <span> </span>.

1.3: HTML Attributes

Attributes can be added to allow us to add more information to an HTML tag, such as a link to another website, or even an image.

Images should be in the same directory as a script. To insert an image, use the format:

<img src="django.png" alt="">

  • Note: This is a type of tag classified as "Self-closing" as it does not require a respective </> at the end.
