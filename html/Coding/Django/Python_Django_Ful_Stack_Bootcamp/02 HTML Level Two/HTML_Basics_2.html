1. Tables:

Tables can be inserted with a <table border=""> tag.

The border thickness can be specified from this tag, or left blank for no border.

There are then 4 main elements within a table:

  • <thead> - indicates the headers of a table, under which all <th> tags go.
  • <th> - indicates header header items, such as Num, State, Capital.
  • <tr> - indicates a new Table row, under which all <td> tags go.
  • <td> = indicates a table cell, which is placed under a specific <tr> tag row.
